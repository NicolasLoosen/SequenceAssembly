Die SequennzeAssembler.jar ausführen mit der frag.txt im selben ordner.
Der Code wurde mit VSCode in java erstellt und sollte für das Verständniss ausreichend kommentiert sein.

Im Menüpunkt Aufgabe 1,2 oder 3 auswählen. Mit 3 wird die 2 automatisch ausgeführt. Und 1 wird natürlich immer mit ausgeführt.
Danach bei Menüpunkten 2,3 noch die editSubValue(1) und die errorRate(10) auswählen dannach sollten die Ausgaben in der Console erscheinen.
